London Economics in 2012

London Economics in London 2012 shows the economic features of London from spatial perspective. D3.js and Google JavaScript API are employed to make this visualization.
What features does it have? Selecting an economic feature from the pull down meanu, you will see each district are colored based on the corresponding value of the selected feature. Also, you can hover over each area to find out more details about it. Since Google map is used as a base map in this visualization, some of the feauters that Google maps are retained, for instance, zoom in/out and street view. The search box at the top right corner allows you to search and find a location of interest.
Appreiciate the students who cleaned the data and made the GeoJson file from DASIL, so I could build this visualization.

Interactive Network of Banks and CDOs(Collateralized Debet Obligation)

Network of Banks and CDOs has been used as a teaching tool in classes to explain economics concepts to students. Users can see the dynamic relatiionship between banks and CDOs by setting parameters.
How does it work? Click on a node to see its directly connected neighbors. The node and its neighbors can be highligted and the other nodes will grey out. Also, a tooltip will pop up when you click on a node, which displays the degree of a node and its neighbors. To understand the impact of a failing node on banks and other nodes, you can set the threshold and select a failing CDO, then effected bank and CDOs will squentially grey out.

2008 Financial Crisis Visualization

2008 Financial Crisis Visualization is an interactive visualization that displays the relationship of banks and CDOs. After learning basic concepts of network analysis and economic, students can go ahead exploring a network with real data. This network can be used to simulate the financial crisis in 2008.
How does it work? Like Network of Banks and CDOs, you can click on a node to see its degree and neighbors. To take a closer look of each node and connection, you can zoom in the network by double clicking on the node of insterest, then the network will zoom in and focus on the selected node.
The datesest is used to build this interactive network is from ProPublica.org.
